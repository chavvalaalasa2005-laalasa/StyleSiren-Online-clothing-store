# 🛍️ StyleSiren – Online Clothing Store

## 📌 Project Overview
StyleSiren is a mini e-commerce web application that allows users to browse clothing products, explore categories, and view items in a simple and user-friendly interface.

This project demonstrates basic front-end web development skills including layout design, navigation, and UI structuring.

---

## 🚀 Features
- 🏠 Home page with featured products
- 🛍️ Browse categories (Men, Women, Kids, Accessories)
- 🔥 View New Arrivals and Best Sellers
- 🛒 Add to Cart functionality (UI-based)
- 👤 Login page interface
- 🔗 Multi-page navigation

---

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (basic)
